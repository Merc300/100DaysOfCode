About

I'm taking an Intro to jQuery course via Microsoft. In the first module of the course, the instructor created an example using Bootstrap buttons to demonstrate how jQeuery is used to select CSS classes by (1)element, (2)class, (3)heirarchy, and (4)attribute. I decided to create my own button example to demonstrate and test my knowledge of concepts I've learned thus far. To clarify, I created heirarchy examplse that demonstrate both parent-child node relationship and parent-descendant relationship.
